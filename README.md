# Simple Paint in SDL with nice effect

## Key Bindings

### General
* v - change layout (background/foreground)
* g - glitch mode for foreground
* SPACE - 
* c - clear all lines
* z, u - undo
* x, b - reverse undo

### Colors
* q - red
* w - green
* e - blue
* r - white
* t - black

### Advanced
* F5 - save drawings to maps/save0.bin
* F9 - load drawings from maps/save0.bin
* F12 - screenshot (bmp format)
